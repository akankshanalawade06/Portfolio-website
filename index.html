Real Time Mask Detection System

We developed the face mask detector model for detecting whether person is wearing a mask or not. Training the model is the first part of this project and testing using webcam using OpenCV is the second part. The dataset consists of 3,833 images with 1,915 images containing images of people wearing masks and 1,918 images with people without masks.
